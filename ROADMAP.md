# 800影视 Web - 开发进度与任务计划

> 最后更新：2026-06-21。本文反映 `frontend/web` 当前实现状态，与 `docs/API_V1.md` 及后端 `/api/v1` 契约对齐情况一致。

---

## API 对接状态

Web 消费端**已完成**与第一方 API 的核心对接：

| 能力 | 端点 / 说明 | 代码入口 |
|------|-------------|----------|
| 业务基址 | `/api/v1` | `src/services/config.js` |
| 认证 | `/api/auth/*`（Cookie 会话） | `src/services/auth.js` |
| 首页 | `/home`、`/recommendations/home`、`/home/curations`；失败回退 `/3/discover` | `src/pages/home.js`、`src/services/api.js` |
| 片库/搜索/详情 | `/3/discover`、`/3/search/*`、`/3/movie\|tv/:id` | `src/services/api.js` |
| 播放量 | `POST /movies/:slug/view` | `recordMovieView()` |
| 取流 | `GET /me/stream/:type/:id.json` | `getStream()` |
| 权限/时长 | `/me/permissions`、`/me/watch-time` | `src/services/permissions.js` |
| 用户数据 | `/me`、`/me/history`、`/me/watchlist`、`/me/watch-later`、`/me/resume` | `src/services/library.js` |
| VIP | `/vip/plans`、`/vip/create-order`、`/vip/order-status` | `src/services/vip.js` |
| 评论/评分 | `/comments/*`、`/ratings/*` | `src/components/comments.js`、`content-rating.js` |
| 创作者/直播 | `/creator/*` | `src/services/creator.js` |

**未接或有意不做（增强项，不阻塞 MVP）：**

- `/me/accounts` 多账号切换
- 搜索页合并创作者专用结果（当前统一走 TMDB `/3/search/*`）
- `/analytics/engagement` 等行为埋点与播放量上报并存，用途不同

**已废弃、代码中不再使用：** `playback-token`、legacy `/api/home`、旧 `/search` / `/rankings`（410，已改走 `/3/*`）。
