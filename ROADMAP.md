## 技术债务清单

| 项目 | 优先级 | 说明 |
|------|--------|------|
| XSS 防护 | 高 | 所有用户输入必须经过 esc() 转义 |
| 错误边界 | 高 | 任何 API 失败不应白屏 |
| 性能监控 | 中 | 添加 Web Vitals 上报 |
| 无障碍 | 中 | 键盘导航、aria 标签、焦点管理 |
| SEO | 低 | 纯 SPA 不需要，但可加 meta 标签 |
| 单元测试 | 低 | 核心模块（signal、router）加测试 |
